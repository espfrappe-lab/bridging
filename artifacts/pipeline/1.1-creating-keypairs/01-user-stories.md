# Stage 1 — User Stories
## Subtopic 1.1: Creating Keypairs
**Source:** `documentation/sdk-usage.md` § "Creating Keypairs"

---

**US-01:** As a developer, I want to generate a random Stellar keypair so that I can create a new account identity with a unique public key and secret seed.

**US-02:** As a developer, I want to create a keypair from an existing secret seed so that I can reconstruct a known Stellar account for signing transactions.

**US-03:** As a developer, I want to create a public-key-only keypair from an account ID so that I can use it for verification or inspection without holding signing capability.
